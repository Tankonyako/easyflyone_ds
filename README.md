# EasyFlyOne - Airline Company Simple Discord Bot
This project is a demonstration of a discord bot! ***You can freely use the materials from this repository*** to implement your features. 

**🎁 Is DLC for original repository: https://github.com/Tankonyako/easyflyone**
## 💡 Functionality:
 - /colors, /about_us, /brand - Brand info about Airline company.
 - /findairport - Find airport by country, IATA code, city, name.
 - /help, /go - Help with bot
 - Loading static data about airlines and etc. from main website!
